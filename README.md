# 📊 Real-Time Cryptocurrency Market Analysis Dashboard Using Power BI

## 📌 Project Overview

This project is an interactive Power BI dashboard that provides real-time cryptocurrency market insights using live data from the CoinGecko API. It enables users to monitor market performance through dynamic visualizations and KPIs.

---

## 🚀 Features

- Live Cryptocurrency Market Data
- Total Market Capitalization KPI
- Total Trading Volume KPI
- Bitcoin Price KPI
- Top Gainer & Top Loser
- Market Capitalization Comparison Chart
- Price Comparison Chart
- Interactive Cryptocurrency Slicer

---

## 🛠️ Technologies Used

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- CoinGecko API
